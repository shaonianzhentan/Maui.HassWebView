Welcome to HassWebView's Internal Web Server!

This page is being served directly from the .NET MAUI application.

You can use this to create custom UI, remote control interfaces, or expose APIs to your web view content.

Available API Endpoints:

  • GET /api/info: Returns system information. Try /api/info?name=Developer.
  • POST /api/echo: Echoes back the JSON message you send.
  • PUT /api/data: Accepts raw data via PUT request.
